# valorant-match-analyzer

Ferramenta para buscar e analisar partidas do Valorant usando a API pública (Henrik API), gerando relatório de performance: K/D ratio, headshot %, mapas mais fortes e agentes mais jogados.

## Motivação
Jogo Valorant competitivamente e queria entender minha performance com dados reais, não só feeling.

## Stack
- Python
- Henrik Dev API (https://docs.henrikdev.xyz)
- Pandas / visualização de dados
